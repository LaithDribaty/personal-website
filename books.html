Books that I recommend in Web learning

HTML & CSS

A good start point in web learning.

Jon Ducket explain HTML and CSS in some manner that he covers all essential concepts. and also give you the ability to expand yourself.

I would recommend it as a start because it explain some basic operations that happens in the internet which is very important to understand how the web is constructed.

Simple language make the book easy for any reader in IT or in Enlgish.

Also the effect of shapes and diagrams and simple drawing in the book makes it attract.

JAVASCRIPT & JQUERY

Just like the previouse book Jon Ducket continues his journy in making great books.

And why javascript is very important to be studied, because that it is the only language that a browser can compile and run, any other language such coffee or typescript are being compiled and build into javascript before deploying them

also JQUERY is a very important javascript library to simplify the code, like their motto "code less do more" and make use of AJAX request effectivly.

Advanced JavaScript

as all other programming languages improves over time to cover increasing demands and cases, javascript released some new versions like ES6

in ES6 many new features were added to javascript, so this book with its examples and homeworks is a very good entry point to learn those new features

also as a plus the book contain a chapter about new technologies that uses javascript to accomplish many useful things

SEO For Dummies

like all for dummies books simple, clear and specific.. the book can be read by specialized or non-specialized becuase it explain how search engines think and how can you do better not just in code but in buisness and commercial.

Books that I recommend in 3D on the web

Learning Blender

When learning technologies or programs we should read the latest released books and courses but for now (2022-4-25) i would recommend this book as a start for blender, the book contains chapters that explain all panels in blender briefly and shows you what can you do with blender.

Also the book gives you information of doing 3d stuff in general and specificly in blender like modeling a human and rigging him and making animation.. eventualy make you confortable with other programs that you might use later.

Simple language and great organizing.. It could be the entry point for the whole world of 3d.

Learning Three.js: The JavaScript 3D Library for WebGL

from basics into everything that you can do in three.js, ofcourse not everything letterly because we may mix some concepts always making new things and wonderful scenes.

the book has github repo that holds examples of each chapter which helps alot later when copy pasting some code fragments.

Books that I recommend in general

Advanced Algorithms And Data Structures

even if there is technologies that help you doing cache system I think that you should somehow know how do cache work..

this book explain so many things that we use everyday in our applications, but actually it explain the real algorithms which we rarly to work with as long as there are apps that provide friendly simple api to use it.

the book covers some hashing algorithms, cache systems, disjoint sets, string processing (trie, radix trie..), multidimensional indexing, Nearest Neighboor Search, clustering, graph theory and genetic algorithms

Dive Into DESIGN PATTERNS

Design pattern is one of the most important concepts to learn in software engineering.. It is the matter of keeping everything organized, maintainable, readable, fast and open to be expanded in the future.

So this book shows us with some meaningful drawing (cute drawing actually) the work flow and cons pros usage and details of every famouse design pattern that you will use in some points of your work.
